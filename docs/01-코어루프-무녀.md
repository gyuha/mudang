# 01. 코어 루프 · 무녀(플레이어)

무녀는 **공격하지 않는다.** 순간순간 플레이어가 다루는 능동 도구는 단 4개다. 이 4개의 상호작용이 게임의 깊이 전부다. (결정 D6, D7)

```
입력(이동/클릭/명령) → 무녀 상태 갱신 → 오라·넉백·전달·모여라 작용 → 동료 생존/화력 변화
```

---

## 1. 입력 추상 레이어 (D5)

플랫폼 분기는 입력 어댑터에서만 일어난다. 게임 로직은 추상 3종만 본다.

| 추상 입력 | 의미 | PC | 모바일 |
|-----------|------|----|--------|
| `move_vector: Vector2` | 무녀 이동 방향(정규화) | WASD / 좌스틱 | 좌측 가상 조이스틱 |
| `aim_point: Vector2` (월드) + `aim_pressed` | 넉백 시전 위치 | 마우스 클릭 좌표 | 화면 탭 좌표 |
| `cmd_rally: bool` | 모여라 트리거 | 스페이스/우클릭 | 화면 우하단 버튼 |

구현: `InputAdapter`(autoload 또는 노드)가 위 3개를 매 프레임 산출 → `Mudang`이 읽는다. 모바일 이식은 이 어댑터 1개만 교체.

```gdscript
# input_adapter.gd  (요지)
class_name InputAdapter extends Node
var move_vector: Vector2
var aim_point: Vector2
var aim_pressed: bool
var rally_pressed: bool
# PC: Input.get_vector(...) / get_global_mouse_position()
# 모바일: 가상 조이스틱 노드 + 화면 터치 → 카메라 unproject
```

## 2. 레버 1 — 오라 (Aura) · 패시브 감속장

무녀를 따라다니는 원형 장(場). 반경 내 적의 이동속도를 곱연산으로 줄인다.

| 파라미터 | 시작값 | 비고 |
|----------|--------|------|
| `aura_radius` | 140 px | 업그레이드로 증가 |
| `slow_multiplier` | 0.6 (40% 감속) | 1.0=무효, 0=정지. 완전 정지 불가 |
| 레벨 보정 | 적 레벨이 무녀보다 높으면 감속률 약화 | DESIGN §8-3 반영. `eff = lerp(slow, 1.0, clamp((enemy_lv-mudang_lv)*0.1, 0, 0.5))` |

- 항상 켜져 있음. 무녀의 포지셔닝이 곧 군중 제어다.
- 구현: 적 이동 계산 시 무녀와의 거리 검사(spatial hash 활용, [06]). 매 적마다 Area2D 쓰지 말 것(500마리 = 폭발).

## 3. 레버 2 — 「물렀거라」 (넉백, Knockback) · 유일한 능동 전투 동사

> 기술명 **「물렀거라」** — 무녀가 잡것을 쫓을 때 외치는 말(코드 식별자는 `knockback`). 대안: 「벽사」「축귀」.

`aim_pressed` 시 `aim_point`를 중심으로 한 원형 임펄스. 적을 바깥으로 밀어낸다. **동료를 지키고 무녀 자신의 살길을 트는 핵심 수단.**

| 파라미터 | 시작값 | 비고 |
|----------|--------|------|
| `knockback_radius` | 90 px | 업그레이드 ↑ |
| `knockback_force` | 380 px (순간 변위 거리 상한) | 업그레이드 ↑ |
| `knockback_cooldown` | 1.2 s | 업그레이드 ↓ |
| 사거리 제한 | 무녀 기준 `max_cast_dist` 320 px | 무제한 시전 방지(밸런스 노브) |
| 적 저항 | 적별 `knockback_resist`(0~1), 보스 ≈ 0.9 | 도깨비/보스는 잘 안 밀림 |

- 데미지 없음. 순수 변위 + 짧은 경직(`0.15s`).
- 시전 위치 중심이라 "동료와 적 사이"를 정확히 노려 떼어내는 에임 스킬이 ceiling.
- 구현: 중심 반경 내 적을 spatial hash로 질의 → `dir = (enemy.pos - center).normalized()` 변위 적용 → `resist` 곱.

```
탭(aim_point) → 쿨다운 체크 → 반경 내 적 질의 → (1-resist) 비례 바깥 변위 + 경직
```

## 4. 레버 3 — 혼불 전달 (Soulfire Transfer) · 공간적 자원 배분

무녀가 보유한 **동료 혼불**을, 동료에게 다가가면 **자동으로** 순차 전달한다. **가까울수록 빠르게** 넘어간다. (D10, 사용자 확정)

규칙(사용자 확정):
- **별도 입력 없음.** `transfer_range` 안에 들어오면 자동 전달 시작.
- 전달 속도는 **거리에 반비례** — 동료에 붙을수록 초당 전달량↑, 범위 가장자리에선 느림.
- 전달 중 거리가 멀어지면(범위 이탈) **즉시 중단**(다시 들어오면 재개).
- 어느 동료에게 먹일지는 플레이어가 "걸어가서/붙어서" 결정 → 빌드 배분의 본질.

전달 속도 공식(시작값):
```
proximity = clamp(1 - dist / transfer_range, 0, 1)   # 붙으면 1, 가장자리면 0
rate = transfer_rate_max * proximity                  # 혼불/s
```

| 파라미터 | 시작값 | 비고 |
|----------|--------|------|
| `transfer_range` | 110 px | 동료 중심, 진입 시 자동 시작 |
| `transfer_rate_max` | 12 혼불/s | 동료에 밀착 시 최대 속도. 업그레이드 ↑ |
| 거리 곡선 | 선형 반비례(위 공식) | 가까울수록 빠름 |
| **전달 시 회복** | 전달량 1당 동료 HP **+0.5**(소폭) | "혼불 먹이면 힐도 됨". 견습무당이 주 힐러, 전달은 부수 회복 |
| 보유 한도 | 무제한(또는 표시용 캡 200) | 디자인 노브 |
| 동시 전달 | 1명만(가장 가까운 동료) | 분산 방지, 의도적 선택 강제 |

> 전달은 **성장(EXP) + 소폭 HP 회복**을 겸한다. 회복량은 견습무당 힐(6/s)보다 작아 역할 침범하지 않음.

상세 흐름은 [03] 혼불 경제 참조.

## 5. 레버 4 — 모여라 (Rally) · 쿨다운 액티브 (D8)

`cmd_rally` 시 발동. 동료들이 **전투를 계속하면서** 무녀 위치로 **서서히** 모인다. 위기 대응·재정비용.

| 파라미터 | 시작값 | 업그레이드 |
|----------|--------|-----------|
| `rally_duration` | 4.0 s | 지속시간 ↑ |
| `rally_cooldown` | 18 s | 쿨 ↓ |
| `rally_pull_bias` | 이동 목표에 무녀 위치를 강하게 가중 | (지속 동안 AI 목표 보정) |

- "영구 상태가 아닌 일시 전술"(DESIGN §8-4) — 지속 끝나면 동료는 자율 AI로 복귀.
- 구현: 지속 동안 각 동료 FSM의 이동 목표 산출에 `무녀 방향` 가중치를 크게 더한다(교전은 유지).

```
모여라 입력 → 쿨다운 체크 → duration 동안 동료 이동목표에 무녀 가중 → 종료 시 자율 복귀
```

## 6. 무녀 성장 (서포터 전용, D7·D11·D12)

무녀 혼불을 주워 EXP가 차면 레벨업 → **게임 일시정지 + 서포터 주술 3택 1.** 공격 주술은 존재하지 않는다.

업그레이드 카테고리(전부 컨트롤/서포트/케어):

| 계열 | 예시 효과 |
|------|----------|
| 생존 | **무녀 최대 HP↑**, 받피 감소, 위기 시 짧은 무적 |
| 오라 | 반경↑, 감속률↑, 레벨 보정 페널티 완화 |
| 「물렀거라」 | 반경↑, 힘↑, 쿨↓, 짧은 스턴 추가, 2회 충전 |
| 혼불 전달 | 전달속도↑, 전달 시 회복량↑/소량 보호막 동반 |
| 모여라 | 지속↑, 쿨↓, 집결 중 동료 받피 감소 |
| 부활/케어 | 채널링 속도↑, 쓰러짐 타이머 연장(넋달래기), 보호막 부여(신령 보호막) |
| 봉인/유틸 | 귀문 봉쇄(범위 적 일시 속박), 정화(장판 제거) |

> DESIGN §11의 "혼불 부적"(공격 주술)은 D7로 폐기. 동료에게 부여하는 강화로 재해석하거나 드롭한다.

레벨업 EXP 곡선(시작값): `exp_to_next(n) = floor(8 * pow(1.15, n-1))`. 튜닝 대상.

## 7. 무녀 자체 능력 (조작 불변값)

| 항목 | 시작값 |
|------|--------|
| 이동속도 | 220 px/s |
| **최대 HP** | **100** (생존 카드로 증가, [09]) |
| 받는 피해 | 적 `contact_damage`를 **접촉당 초당**으로 받음(거점·동료와 동일 모델). 오라 감속 + 「물렀거라」로 군집 회피가 곧 생존 |
| 충돌 | 적과 물리 충돌은 없으나(통과) **접촉 중 피격 판정 있음** |
| 회복 수단 | 견습무당 힐, 혼불 전달 부수 회복([§4]), 정화/보호막 카드. 시간 자동 재생은 없음(노브) |

> **결정 D6-a(갱신):** 무녀에게 **체력이 있다.** **무녀 사망 = 즉시 패배** (주 패배 조건). 거점 파괴도 패배. 동료 전원 상실은 화력 0의 사실상 게임오버이나 공식 패배 트리거는 무녀/거점. → "나도 살고 동료도 지킨다"는 이중 압박이 포지셔닝 긴장을 만든다.
