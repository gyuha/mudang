# 12. 동료 직업 로스터 · 설정

> 이 문서는 **D16 해금 동료 풀**(출전 슬롯 2→4)의 구체화다.
> 게임에 등장하는 **동료(직업)들의 세계관·성격·전투 역할 설정**을 모은 로스터 바이블이며, 동시에 각 직업을 **실제 구현 모델**(단일 `CompanionDef` + `role_id` 분기, D25)에 어떻게 얹는지를 명시한다.
> 컨셉은 `DESIGN.md`, 결정의 권위는 [00] 묶음. 충돌 시 [00]이 우선한다.

---

## 0. 무녀는 직업이 아니다 (범위 못박기)

이 로스터에 **무녀는 없다.** 무녀는 플레이어 본인이며 **무공격 순수 서포터**다(D6·D7). "부적·굿으로 동료를 끌어내고 전장을 지휘한다"는 무녀의 정체성은 직업 슬롯이 아니라 **게임의 조작 모델 그 자체**다 → [01] 코어 루프 참조.

따라서 이 문서가 다루는 직업은 전부 **동료(`Companion`)** — 자율 AI 전투원이고, 모든 화력의 원천이다(D9). 플레이어는 동료의 스킬을 직접 쏘지 않는다. 동료는 **역할대로 알아서 싸우고**, 무녀는 그들을 **살리고·배치하고·먹인다.**

> 결과적으로 직업의 "개성"은 *플레이어가 누르는 버튼*이 아니라 **자율 거동 성향 + 세계관 + 무녀와의 관계**로 드러난다. 화려한 능동 기믹(덫·소환·분신 등)은 현 모델 밖이며 §4에 별도로 격리한다.

---

## 1. 로스터 한눈에 — 역할 매핑

모든 동료는 단일 `CompanionDef`의 인스턴스이며 `role_id`로 거동이 갈린다(tank / ranged / healer / aoe). 한 역할에 **여러 직업이 공존**할 수 있다(같은 role_id, 다른 스탯·세계관·연출). 엔진 변경 없이 .tres만 추가하면 된다.

| role_id | 역할 | 구현됨(1장 슬라이스) | 후속 해금 후보 |
|---------|------|---------------------|----------------|
| `tank` | 전열·도발·받피 흡수 | **화랑 호위병** ([09]) | 착호갑사 |
| `ranged` | 원거리 단일 화력·카이팅 | **활잡이** ([09]) | 도사 · 포수 · 추노객 |
| `healer` | 회복·정화·보호 | **견습 무당** ([09]) | 승병 · 의원 |
| `aoe` | 군집 제압·광역 제어 | (탈 쓴 퇴마사 — 예정, [02]§2.4) | 광대 · 장례사 |
| *(미정)* | **근접 고속 딜러 — 현 4역할에 없음** | — | **무예별감** (신규 거동 필요, §3.5·§4) |

**상태 범례:** `구현됨` = 슬라이스에 실제 데이터/구현 존재 / `후속` = 세계관·역할 매핑 확정, 데이터는 후속 마일스톤 / `예약` = 고유 기믹이 현 엔진 모델 밖이라 엔진 작업 선행 필요(§4).

> **⚠️ 시대 정합성 메모(미결):** 구현된 탱커 **화랑 호위병**의 "화랑(花郞)"은 **신라**의 청년 무사단이고, 이 로스터의 신규 직업은 전부 **조선**이다. 현재는 의도적으로 **공존**시킨다(이미 커밋된 슬라이스 데이터를 함부로 리네이밍하지 않음 — 수술적 변경 원칙). 시대를 통일하려면 후속에 **화랑 호위병 → 착호갑사**로 흡수/리스킨하는 결정(미래 D-항목)이 필요하다. **이 문서가 임의로 바꾸지 않는다.** (D26)

## 2. 해금 → 편성 → 출전 흐름 (D16)

직업은 영구메타로 해금되고, 런 시작 전 편성 화면에서 슬롯에 채운다. 슬롯은 시작 2개, 메타로 최대 4개까지 늘린다.

```
스테이지 클리어 → 동료 해금(영구메타) → 해금 풀에 추가
   → [런 시작 전] 출전 편성: 풀에서 슬롯 수만큼 선택(시작 2 → 최대 4)
   → 출전 → 런 동안 자율 전투 → (쓰러짐/부활은 무녀 책임, [02])
```

편성의 본질은 **역할 조합**이다. 탱 없이 원딜만 셋이면 전열이 무너지고, 힐 없이 가면 케어 부담이 무녀에게 쏠린다. 직업이 많아질수록 "어떤 역할 3~4개를 데려갈까"의 빌드 결정이 깊어진다. 상세 편성 규칙은 [03] 혼불 경제·성장·메타 참조.

## 3. 직업별 상세 설정

각 카드 형식: **정체성 → 내력(세계관) → 성격 → 무녀와의 관계 → 대표 무기 → 전투 역할(role + `CompanionDef` 매핑 + 시그니처) → 레벨업 카드 방향 → 상태.**
모든 수치 언급은 **시작값/방향성**이며 코드 상수가 아니라 데이터(.tres)로 노출한다([05]). 슬라이스 3종(화랑 호위병·활잡이·견습 무당)의 구체 수치는 [09]가 단일 권위이므로 여기 중복하지 않는다.

---

### 3.1 착호갑사 (捉虎甲士) — `tank` · 후속

- **정체성:** 중갑을 두른 요괴 사냥 전문 군인. 전열에서 적을 붙잡고 동료와 무녀가 설 자리를 만든다.
- **내력:** 본래 맹수(특히 호랑이)를 잡던 관군 부대. 밤마다 인가를 덮치는 것이 짐승이 아니라 **잡것**임이 드러나며, 활인서의 밤을 지키는 척귀(斥鬼) 부대로 재편됐다.
- **성격:** 과묵하고 우직하다. 말보다 몸을 먼저 던진다. 무녀의 지시를 의심 없이 따르되, 위험할 때 가장 먼저 무녀 앞을 막아선다.
- **무녀와의 관계:** 호위 무사. "무당님은 굿만 하시오, 칼은 내가 맞겠소." 무녀가 가장 신뢰하는 방패.
- **대표 무기:** 장창, 환도, 월도, 쇠사슬.
- **전투 역할:** `role_id=tank`. `taunt_radius` 큼(주변 적 도발 → 동료 대신 자신을 노리게), `damage_reduction` 높음, `kite_min=0`(카이팅 없음). 화랑 호위병과 같은 역할의 **더 단단한 변종**(받피 흡수·도발 반경 우위, 기동 둔함).
  - *시그니처(현 모델로 표현 가능):* 도발 반경 + 받피 감소 수치를 화랑보다 공격적으로. 광역 베기(부채꼴).
  - *시그니처(예약):* 쇠사슬로 적을 **끌어당김**(역넉백) — 현 「물렀거라」가 미는 것의 반대. 끌기 거동은 신규(§4).
- **레벨업 카드 방향:** 도발 반경↑ / 받피−% / 광역 베기 / 최대HP↑ / (예약)쇠사슬 견인.
- **상태:** 후속. 시대 정합성상 화랑 호위병의 후계/흡수 1순위 후보(§1 메모).

### 3.2 도사 (道士) — `ranged` · 후속

- **정체성:** 오행·음양·방술을 다루는 정통 술법사. 속성 도술로 원거리에서 적을 친다.
- **내력:** 산중에서 수십 년 도를 닦다 세상의 기운이 탁해짐을 읽고 하산했다. 부적과 호리병에 잡것을 가두는 봉마(封魔)의 전문가.
- **성격:** 초연하고 깐깐하다. 무녀의 굿을 "민간의 잡술"이라며 처음엔 얕보지만, 동료를 살리는 그 손길에서 자신에게 없는 길을 본다.
- **무녀와의 관계:** 묘한 라이벌이자 스승 격. 술법의 이론은 도사가, 사람을 묶는 정(情)은 무녀가 앞선다.
- **대표 무기:** 목검, 부적, 호리병, 지팡이.
- **전투 역할:** `role_id=ranged`. 활잡이와 같은 원거리 역할이되 **속성 술법형**: `kite_min>0`(접근 시 후퇴 사격), `target_priority=DENSEST 또는 ELITE`. 화염·번개 도술 = 시각/연출 차별화, 기계적으로는 관통/광역 투사체로 근사.
  - *시그니처(현 모델):* 투사체가 **관통 + 소폭 광역**(활잡이의 단일 관통과 차별).
  - *시그니처(예약):* 결계 설치(길목 차단 장판) — 설치형 지속 엔티티 필요(§4).
- **레벨업 카드 방향:** 관통↑ / 광역 반경↑ / 공격주기↓ / (예약)결계.
- **상태:** 후속.

### 3.3 승병 (僧兵) — `healer` · 후속

- **정체성:** 불법(佛法)과 법구로 원혼·사귀를 정화하는 무승. 회복·해제·대언데드의 기둥.
- **내력:** 임진의 난리 때 창을 들었던 의승군의 후예. 칼보다 염불이 잡것에게 더 아프다는 것을 안다.
- **성격:** 자비롭되 단호하다. 적의 고통(원혼)을 가엾이 여겨 "성불(成佛)시킨다"고 표현한다. 동료의 죽음을 가장 무겁게 받아들인다.
- **무녀와의 관계:** 신앙의 결은 다르나(불교 vs 무속) 같은 적을 본다. 무녀의 부활 채널링을 "넋을 붙드는 일"이라며 돕는다.
- **대표 무기:** 육환장, 염주, 금강저, 월도.
- **전투 역할:** `role_id=healer`. 견습 무당과 같은 힐 역할이되 **단단한 정화형**: `heal_per_sec>0`, `damage_reduction`도 소폭(맷집 있는 힐러). 정화(장판/디버프 제거) 우위, 원혼·언데드 계열 적에 보조딜 가산.
  - *시그니처(현 모델):* 정화 쿨 우위 + 자기 받피 감소 + 대(對)원혼 추가 피해(적 태그 기반).
  - *시그니처(예약):* 금강결계(아군 광역 보호막 존), 업보 누적 적에 추가 피해 — 적 상태 추적 필요(§4).
- **레벨업 카드 방향:** 힐량↑ / 정화 쿨↓ / 자기 받피−% / 대원혼 추가피해 / (예약)금강결계.
- **상태:** 후속.

### 3.4 추노객 (推奴客) — `ranged` · 후속(함정은 예약)

- **정체성:** 도망친 자를 쫓던 추적꾼. 이제는 인간 세상에 숨어든 둔갑 요괴를 사냥한다. 단일 정예 처치 + 함정.
- **내력:** 발자국과 냄새만으로 천 리를 쫓는 사냥꾼. 사람으로 둔갑한 잡것을 가려내는 눈을 가졌다.
- **성격:** 냉소적이고 실리적. 돈과 표식으로 움직이는 듯하나, 무녀 곁에 머무는 진짜 이유는 말하지 않는다.
- **무녀와의 관계:** 거래로 시작된 동행. 무녀는 그가 쫓는 "흔적"의 정체를 알고도 모른 척한다.
- **대표 무기:** 활, 쇠뇌, 단도, 포승줄.
- **전투 역할:** `role_id=ranged`. 원거리 역할이되 **정예 처치 특화**: `target_priority=ELITE`(도깨비·정예 우선), 단일 고화력. 활잡이가 군중 정리라면 추노객은 굵은 한 발.
  - *시그니처(현 모델):* 정예 대상 추가 피해, 첫 일격(표식) 보너스.
  - *시그니처(예약):* 덫/올가미 설치(밟으면 속박) — 설치형 트리거 엔티티 필요(§4).
- **레벨업 카드 방향:** 정예 추가피해↑ / 표식 일격↑ / 관통 / (예약)덫 설치.
- **상태:** 후속(함정 기믹은 예약).

### 3.5 무예별감 (武藝別監) — *역할 미정* · 후속(신규 거동 필요)

- **정체성:** 궁중·관아가 비밀리에 양성한 요괴 토벌 무사. 빠른 발도와 연속 베기로 적 사이를 누빈다.
- **내력:** 별운검(別雲劍)의 그림자 조직. 왕실 비전 무예를 익혔고, 공식 기록엔 없는 존재다.
- **성격:** 자부심 강하고 날카롭다. "지킨다"보다 "벤다". 무녀의 무공격 원칙을 답답해하면서도 그 판단을 따른다.
- **무녀와의 관계:** 칼끝의 충성. 무녀가 가리키는 곳을 벤다.
- **대표 무기:** 환도, 쌍검, 편곤.
- **전투 역할:** **근접 고속 딜러** — 현 4역할(tank/ranged/healer/aoe) 어디에도 깨끗이 안 맞는다. 탱커의 근접이지만 **받피 흡수·도발 없이 고DPS·고기동**이다.
  - *처리 방향(택1, 후속 결정):* ① `tank` 거동에서 `taunt_radius=0`·`damage_reduction=0`·공격력↑·이동↑로 **튜닝만으로 근사**(가장 싼 길), 또는 ② 신규 거동 `role_id=melee_dps`(적 사이 이동·반격) 추가(§4).
  - *시그니처(예약):* 회피 직후 반격, 적 사이 대시 베기.
- **레벨업 카드 방향:** 공격주기↓ / 이동↑ / 연속 베기 단수↑ / (예약)반격.
- **상태:** 후속. **역할 공백 — 억지 매핑하지 않고 신규 거동 결정 전까지 보류.**

### 3.6 포수 (砲手) — `ranged` · 후속(특수탄은 예약)

- **정체성:** 조총과 화약을 다루는 강력한 원거리 직업. 갑주·비늘을 관통하는 단발 폭딜, 거대 요괴·보스 담당.
- **내력:** 북방·산포수 출신의 명사수. 한 발의 무게를 안다. 재장전이 곧 목숨값이다.
- **성격:** 무뚝뚝하고 신중하다. 말이 적고 방아쇠가 정직하다. 도깨비·보스 앞에서 가장 빛난다.
- **무녀와의 관계:** "거리는 내가 벌어주겠소." 무녀가 화력 게이트(보스)를 넘게 하는 열쇠.
- **대표 무기:** 조총, 승자총통, 비격진천뢰.
- **전투 역할:** `role_id=ranged`. **단발 고화력 + 재장전 취약** 변종: 낮은 `attack_period`의 반대 — **긴 주기·큰 피해**, 재장전 동안 무방비(카이팅으로 시간 벌기, `kite_min>0`). `target_priority=ELITE`(보스/거대 요괴).
  - *시그니처(현 모델):* 단발 초고피해 + 재장전 창(취약 구간) → 무녀의 보호 대상화.
  - *시그니처(예약):* 은탄·소금탄·부적탄 등 특수탄 전환(속성), 화약통/지뢰 설치 — 탄종/속성 시스템 + 설치물 필요(§4, [11] 투사체 확장과 연계).
- **레벨업 카드 방향:** 단발 피해↑ / 재장전 시간↓ / 대보스 추가피해 / (예약)특수탄.
- **상태:** 후속(특수탄·설치는 예약).

### 3.7 의원 (醫員) — `healer` · 후속(독·디버프는 근사/예약)

- **정체성:** 약초·침술의 전투 의술가. 동료를 강화하고, 요괴의 약점을 짚는다.
- **내력:** 활인서(이 장의 무대!)의 의원. 사람을 살리던 손으로 잡것의 신체를 해부해 약점을 찾는다. 이 직업은 1장 무대와 서사적으로 가장 밀착한다.
- **성격:** 차분하고 분석적. 요괴조차 "증상과 부위"로 본다. 무녀의 직관과 의원의 관찰이 보완 관계.
- **무녀와의 관계:** 활인서의 동지. 무녀가 넋을 다루면 의원은 몸을 다룬다.
- **대표 무기:** 침, 약낭, 단도, 독연막.
- **전투 역할:** `role_id=healer`. 견습 무당의 회복과 차별 — **강화(버프)·약점 노출형**: `heal_per_sec>0` + 아군 강화(보호막/소폭 버프). 침술 = 동료 일시 강화.
  - *시그니처(현 모델):* 보호막 부여, 회복.
  - *시그니처(예약):* 독침/마비독(적 디버프 — 현 모델에 적 디버프 적용 거동 없음), 해부로 약점 노출(적 받피 증가) → 적 상태/디버프 시스템 필요(§4).
- **레벨업 카드 방향:** 보호막량↑ / 힐량↑ / (근사)동료 공격력 소폭 버프 / (예약)독·약점.
- **상태:** 후속(회복·보호는 근사 가능, 독·디버프는 예약).

### 3.8 광대 (廣大) — `aoe` · 후속(환술 대부분 예약)

- **정체성:** 탈춤과 재담으로 요괴를 속이고 흔드는 교란꾼. 군집 제어·도발·아군 버프.
- **내력:** 떠돌이 재인(才人) 패거리의 우두머리. 탈 속에 신(神)이 깃든다 믿으며, 장단으로 산 자와 죽은 자를 모두 어른다.
- **성격:** 능청맞고 종잡을 수 없다. 농담 뒤에 슬픔을 감춘다. 무녀의 엄숙함과 정반대라 오히려 합이 맞는다.
- **무녀와의 관계:** 굿판의 짝. 무녀가 신을 부르면 광대가 분위기를 만든다. 가장 인간적인 동료.
- **대표 무기:** 탈, 부채, 장구, 줄타기 도구.
- **전투 역할:** `role_id=aoe`. 탈 쓴 퇴마사([02]§2.4)와 같은 광역 제압 계열의 **교란·버프형**: 가장 밀집한 군집 중심 타게팅. 도발(`taunt`)을 비(非)탱이 쓰는 드문 케이스.
  - *시그니처(현 모델):* 광역 도발(군집 어그로 끌기).
  - *시그니처(예약):* 분신·환영(가짜 동료로 어그로 흡수), 탈 교체로 전투 형태 전환, 장단 버프(아군 공속↑) → 더미 유닛·동적 역할 전환·아군 버프 거동 필요(§4).
- **레벨업 카드 방향:** 도발 반경↑ / 도발 지속↑ / (예약)분신·버프.
- **상태:** 후속(도발 근사 가능, 환술 핵심은 예약). aoe 슬롯은 슬라이스에 아직 없음 → 탈 쓴 퇴마사 다음 순위.

### 3.9 장례사 (염사, 殮師) — `aoe` · 예약(사령술 충돌 큼)

- **정체성:** 상여·염(殮)·묘지의 요소를 결합한 어두운 직업. 혼령·시체·죽음을 다룬다.
- **내력:** 죽은 자를 거두는 일을 업으로 삼다 산 자와 죽은 자의 경계를 넘은 자. 원혼을 성불시키거나 부린다.
- **성격:** 음울하고 과묵하다. 죽음을 두려워하지 않아 가장 위험한 곳에 선다. 무녀에게 "당신도 결국 우리 편(저승)"이라 농 같은 진담을 던진다.
- **무녀와의 관계:** 가장 무거운 동료. 무녀의 부활(넋 붙들기)과 장례사의 사령술은 동전의 양면 — 협력이자 윤리적 긴장.
- **대표 무기:** 상여 방울, 삽, 만장, 혼백등.
- **전투 역할:** `role_id=aoe`(혼령 장판 지속피해로 근사). **하지만 정체성의 핵심인 사령술은 현 모델과 충돌이 가장 크다.**
  - *시그니처(현 모델로 근사 가능):* 혼령 장판(범위 지속 피해존).
  - *시그니처(예약·충돌 큼):* 원혼 사역/소환(소환물 = 또 다른 자율 AI 유닛 → 동시 유닛 수 폭증, 성능 D19와 정면 충돌), 시체 폭발·시체 방패(적 시체 상태 추적) → §4.
- **레벨업 카드 방향:** 장판 피해↑ / 장판 반경↑ / (예약)사령·시체.
- **상태:** **예약.** 가장 매력적이지만 가장 비싸다. 소환·시체 시스템 결정 전까지 데이터화하지 않는다.

## 4. 예약 기믹 정리 (현 모델 밖 — 엔진 작업 선행 필요)

아래는 직업 판타지엔 좋지만 **"자율 AI + 단일 `CompanionDef`"(D9·D25) 모델로는 공짜가 아닌** 기믹이다. 각각 엔진 확장이 선행돼야 하며, **본 로스터 문서는 이것들을 데이터로 약속하지 않는다.** 후속 마일스톤에서 개별 결정 대상.

| 기믹 | 출처 직업 | 현 모델 충돌 지점 | 근사 대안 | 필요한 엔진 작업 | 후속 우선순위 |
|------|-----------|------------------|-----------|------------------|---------------|
| 설치형 덫/올가미 | 추노객 | 동료는 "공격"만, 월드에 지속 설치물 없음 | 표식 일격으로 일부 대체 | 설치형 트리거 엔티티 + 수명/판정 | 중 |
| 결계/봉인 장판 | 도사·승병 | 아군용 지속 장판 부재 | 무녀 카드 "귀문 봉쇄"([09]§4)로 일부 흡수 | 정적 해저드존(D22)의 **아군 버전** | 중 |
| 특수탄/탄종 전환 | 포수 | 동료 투사체가 단일 속성 | 단발 고피해로 대체 | 투사체 속성/탄종 시스템([11] 연계) | 중 |
| 적 디버프(독·마비·약점) | 의원 | 동료가 적에 상태이상을 거는 거동 없음 | 보스 텔레그래프·정화는 있음 | 적 상태이상 컴포넌트 + 적용 거동 | 하 |
| 분신/환영 | 광대 | 가짜 동료 = 추가 유닛·타게팅 가중 | 광역 도발로 대체 | 더미 유닛 + 어그로 분산 로직 | 하 |
| 동적 역할 전환(탈 교체) | 광대 | `role_id` 런타임 고정 가정 | — | 런타임 role 스왑 + UI | 하 |
| 소환물(원혼 사역) | 장례사 | **소환 유닛 = 또 다른 자율 AI** | — | 소환 유닛 풀 + AI + 성능(D19 재검토) | 보류 |
| 시체 폭발/시체 방패 | 장례사 | 적 시체 상태 추적 부재 | — | 시체 엔티티 상태머신 | 보류 |
| 견인(끌어당김) | 착호갑사 | 「물렀거라」는 밀기만 | — | 역방향 임펄스(상대적으로 저렴) | 중상 |
| 근접 고속 딜러 거동 | 무예별감 | 4역할에 melee_dps 없음 | tank 튜닝으로 근사(§3.5①) | (선택) 신규 `role_id=melee_dps` | 중 |

> 원칙: **근사 가능한 것은 기존 필드/카드로**, 근사 불가능한 것은 **예약**으로 미룬다. 어떤 기믹도 "있어 보이니까" 엔진에 넣지 않는다(YAGNI·D8·D25).

## 5. 정합성 메모 & 구현 체크리스트

**정합성:**
- 무녀는 직업 아님 → 로스터에서 제외, [01] 단일 권위(§0). `[높음]`
- 직업의 개성은 능동 버튼이 아니라 **자율 거동 + 설정 + 무녀 관계**로 표현 → D6·D9 유지. `[높음]`
- 한 역할에 다중 직업 공존은 `CompanionDef` 다중 인스턴스로 엔진 변경 없이 가능 → D25 유지. `[높음]`
- 화랑(신라) vs 신규 조선 직업의 시대 불일치는 **미결**로 남김. 통일 시 화랑 호위병→착호갑사 흡수가 1순위(§1 ⚠️, D26). `[중간]`
